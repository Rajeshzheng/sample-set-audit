For Upasna ‚Äî Setting up the progress monitoring dashboard

Account: Log in to Google Sheets with the AIGC Gmail account. Find the sheet named "Progress Monitor".

What you'll be doing

The raw_log tab is already receiving real-time data from annotators. You just need to set up the dashboard tab with formulas so we can see everyone's progress at a glance.

4 Paste formulas ‚Äî click each box below to copy, then paste into the cell

Click on cell C2, then paste:

=IFERROR(INDEX(raw_log!E:E,MATCH(2,1/(raw_log!B:B=A2)/(raw_log!D:D="780"),1)),0)

Click on cell D2, then paste:

=IFERROR(INDEX(raw_log!F:F,MATCH(2,1/(raw_log!B:B=A2)/(raw_log!D:D="780"),1)),0)

Click on cell E2, then paste:

=IF(D2>0,TEXT(C2/D2,"0.0%"),"-")

Click on cell F2, then paste:

=IFERROR(INDEX(raw_log!E:E,MATCH(2,1/(raw_log!B:B=A2)/(raw_log!D:D="1000"),1)),0)

Click on cell G2, then paste:

=IFERROR(INDEX(raw_log!F:F,MATCH(2,1/(raw_log!B:B=A2)/(raw_log!D:D="1000"),1)),0)

Click on cell H2, then paste:

=IF(G2>0,TEXT(F2/G2,"0.0%"),"-")

Click on cell I2, then paste:

=IFERROR(TEXT(INDEX(raw_log!A:A,MATCH(2,1/(raw_log!B:B=A2),1)),"MM/DD HH:mm"),"-")

5 Copy Row 2 formulas down to Row 6

Select cells C2 to I2 ‚Üí Copy (Ctrl+C) ‚Üí Select C3 to I6 ‚Üí Paste (Ctrl+V)

The formulas will auto-adjust for each person's name.

6 Done! Verify

If annotators have already refreshed their pages, you should see numbers appearing. If they haven't started yet, you'll see 0 and - which is normal.

Important: The formulas in C, D, F, G, I columns are array formulas. If Google Sheets shows a popup asking about array formulas, click OK or Allow.

If anything looks wrong, message Rajesh. Do not edit the raw_log tab ‚Äî it receives data automatically.
